# Canary-DVB-T 📡

`TVHeadend Mux Files for Canary Islands`

ℹ️ This repo maintains the mux files for the canary islands updated to be used in TVHeadend installations.

- `get_channels.py` Command to download all updated data from `www.muxtdt.com`.
Please note that this utility was written in a hurry and was not well planned so the result is quite a spaghetti-code. Even though... I will refactor it when I have a little spare time. Or if it urges to you, the you can do it yourself. I will happily accept any PR in that sense.
